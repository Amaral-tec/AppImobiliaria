Aplicação Java WEB com JSP

Aplicação desenvolvida usando o padrão de arquitetura Microservices
Desenvolvi uma camada de adaptadores que irá ligar com o framework 
Estou trabalhando com abstrações para então que dentro do core da aplicação não dependenda haja dependencias de elementos externos

--- Paradigma: Orientado a Objetos
--- Padrão de Projeto: Abstract Factory
--- Modelo de Arquitetura: Microservices
--- Cobertura de testes com JUnit 4

Para o desenvolvimentos foram utilizado:

- Bootstrap 
- CSS3
- JavaScript
- Java

Banco de Dados:

- MySql

Servidor

- Local: XAMPP v 3.3.0
- Web: Apache Tomcat v 9.0.65

Libs

- apache-tomcat-9.0.65
- commons-fileupload-1.3.1
- commons-io-2.4
- mysql-connector-java-8.0.18
- JUnit 4.13.2
